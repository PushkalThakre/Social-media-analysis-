# Social-media-analysis-
A data analysis project that evaluates social media engagement across platforms by analyzing likes, comments, shares, and views to identify high-performing content types and optimal posting times.

Overview-
This project analyzes social media post performance to understand engagement patterns across different platforms. The analysis focuses on identifying high-performing content types and the best posting times to maximize audience interaction.

Tools & Technologies-
Python
Pandas
Matplotlib
Seaborn
Jupyter Notebook

Key Analysis-
Engagement analysis using likes, comments, and shares
Content type performance (images, videos, text)
Platform engagement comparison
Optimal posting time identification

Insights-
The analysis shows that video and short-form content generate higher engagement, while evening posting times lead to increased user interaction.

Purpose-
The project demonstrates how data analytics can help optimize social media marketing strategies and audience engagement.
